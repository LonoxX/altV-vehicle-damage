# Usage

## Requirements
-   Alt:V

## Installation & First Start

1. Copy the contents of this repo to `altv-server\resources\altv-vehicle-damage`
3. Start server
4. Have fun crashing!

## Modify

You can easily exclude any Vehicle Classes in Air Movement Control and Damage.
